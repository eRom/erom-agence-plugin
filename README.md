# Agence Plugin - Caserne


## Claude Code

```
/plugin marketplace add eRom/erom-marketplace
```

```
/plugin install caserne@erom-marketplace
```


## Codex CLI

Open the plugin search interface:
```
/plugins
```

Chosse [Add Marketplace] -> `eRom/erom-marketplace` -> `caserne`


## Codex App

* In the Codex app, click on Plugins in the sidebar.
* Click on 'Manage' and add `eRom/erom-marketplace` marketplace.
* Select `caserne` plugin.


## Antigravity / AGY

```
agy plugin install https://github.com/eRom/erom-agence-plugin
```


## Cursor

```
/add-plugin eRom/erom-marketplace
```
